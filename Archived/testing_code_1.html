The data set has 16 attributes. With the exception of the 'name' and 'current tempo' attributes, all the data attributes are derived from the frequency and location information from the music score. A description of the data attributes can be seen below

    1. Location:

    Location is a measure of time and describes how many quarternotes any particular note is from the beginning a particular solo

    2. Note:

    The midi frequency of a each note in the solo.

    3. Part:

    The score originally contained 5 parts (one for the solo, one for the fifth, seventh, third and root. Although it is not used in the analysis it has been kept in the data set for future work where analysis is across a number of parts

    4. Root:

    The midi frequency of the root note of chord over which the solo note is being played

    5. Third:

    The midi frequency of the third chord over which the solo note is being played


    6. Fifth:

    The midi frequency of the fifth of the chord over which the solo note is being played

    7. Seventh:

    The midi frequency of the seventh of chord over which the solo note is being played

    8. Measure:

    The measure (and exact location within that measure) of any given note in a solo

    9. Prep:

    The distance between midi frequency of a note in a row of data, and the note in the previous row of data

    10. Res:

    The distance between midi frequency of a note in a row of data, and the note in the next row of data

    11. P_offset:

    This is a measure of time, being distance in quarter notes between a note in a row or data, and the note in the previous row of data. It is used to ascertain whether the previous note is part of same phrase as the currrent note

    12. R_offset:

    This is a measure of time, being distance in quarter notes between a note in a row or data, and the note in a note in the row of data. It is used to ascertain whether the next note is part of same phrase as the currrent note

    13. Chord Type:

    Calculated by examining the distance between the frequencies of the root, third, fifth and seventh and assigning an appropriately named chord

    14. Dist_note_to_root:

    The distance between the midi frequency of the note of the solo and the midi frequency of the root. This is an absolute value and calculated modulo 12

    15. Current Tempo

    An approximation, calculated from the recording

    16. Name:

    The name of the song from which the solo is taken

The data set is also indexed (0:7858) and each record is given a sequential number. This is a default of the Data Frame structure but also allows analysis of how melody construction changs over time. The order of the data set also reflects the order in which these pieces were recorded so each record refers to a later point in time

3.1 General Remarks About The Data Set

There are 7858 rows of data in the data set which is the total number of notes in all solos. Even at this high level, there are valuable insights can be gained. Although at any one time in any given solo, there are 88 choices (being the notes of the piano), only 48 of these are utilised, and some are utilised far more than others. Tables 3.1 and 3.2 (being a statistical summary of 3.1) show that notes in the midi frequencies between 60 (being middle C4) and 75 (Eb5) are far more predominant than the other notes. While the upper range of the piano is used extensively, note choices for the solo are never below the Bb2

As well as examing occurences of discrete frequencies, it is also possible to look at the notes themselves, regardless of the octave in which they appear. This data (seen in Figure 3.1) shows that there are some midi frequencies (such as Eb, C, G and D) which occur more approximately twice as much as other notes (B, F#, E and C#)

One hypothesis to explain this may be that it is simply due to the choice of tonality jazz standards usually employ (such as the tonality of F Major, Bb Major etc.). Key centres of, for example, B Major and E major are less common in jazz. This suggests there is some kind of relationship between the notes in solos and the notes of the harmony. Anecdotally this is the case, but its worth noting that the data supports this. Figure 3.2 shows a count of how frequenctly each note in the octave occurs in the harmony and there is a relationship that can be seen between the commonly occuring notes in the harmony and the commonly occuring notes in the solo
